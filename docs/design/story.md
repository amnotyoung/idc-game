# Aid World — Story Design

## 핵심 메시지
> 우리가 협력하면 뭔가 이룰 수 있다.

## 타깃 오디언스
일반인 (ODA 비전문가). 전문 용어는 게임 내에서 자연스럽게 설명.

## 플레이타임
총 2시간 이내 (챕터당 20~25분)

---

## 메인 사업 설정

**"나이탬바 섬 재생에너지 전환 사업"**

디젤 발전에만 의존하던 외딴 섬을 태양광으로 전환하는 KODA 사업.
전임자가 갑자기 귀국하면서 주인공이 중간에 이어받는다.

---

## 전체 스토리 아크

```
발령 → 혼자 해결 시도 → 벽에 부딪힘 →
억지로 손 내밀기 → 협력의 첫 성과 →
위기 (각자도생의 유혹) → 신뢰로 극복 → 완료
```

---

## 챕터별 설계

### 1장 — 수바: "인수인계"
- **장소**: KODA 피지 사무소, 수바 시내
- **시간**: 20분
- **등장 NPC**: Mere (첫 만남), James (첫 만남)

| 구분 | 내용 |
|------|------|
| 상황 | 전임자가 남긴 메모 한 장이 전부. 사업은 6개월째 제자리. |
| 목표 | 현재 상황 파악, 첫 이해관계자 만남 |
| 핵심 장면 | Mere가 사무소에 찾아와 따진다: "또 새 담당자예요? 몇 번째야" |
| 플레이어 선택 | 방어적으로 대응 vs 솔직하게 인정 |
| 챕터 결말 | 정부 승인 + 커뮤니티 반발, 두 가지가 동시에 막혀있음을 파악 |
| 메시지 복선 | 주인공 혼자 파악할 수 있는 정보의 한계가 드러남 |

---

### 2장 — 수바: "서류와 사람"
- **장소**: 피지 국가계획부, 회의실
- **시간**: 20분
- **등장 NPC**: Timoci (메인), James (조연)

| 구분 | 내용 |
|------|------|
| 상황 | 사업 승인이 Timoci 책상에서 3개월째 멈춰있음 |
| 목표 | 정부 승인 획득 |
| 핵심 장면 | Timoci는 "서류 보완 필요"를 반복. 실제론 정치적 눈치보기. |
| 플레이어 선택 | 압박 vs 체면 살려주기 vs James에게 도움 요청 |
| 함정 선택지 | James에게 먼저 해결해달라 하면 → 나중에 James가 공동 성과를 요구함 |
| 챕터 결말 | 승인은 받지만 조건이 붙음: "커뮤니티 동의서 필요" |
| 메시지 복선 | 제도는 사람이 움직인다 |

---

### 3장 — 나우소리: "의식과 신뢰"
- **장소**: 농촌 마을, 추장 집
- **시간**: 25분
- **등장 NPC**: Ratu Josefa (메인), Mere (핵심 조력자)

| 구분 | 내용 |
|------|------|
| 상황 | 나이탬바 섬으로 가려면 Ratu Josefa의 허가가 필요 |
| 목표 | 세부세부 의식 통과, 커뮤니티 동의 획득 |
| 핵심 장면 1 | 세부세부 의식 미니게임 — 절차를 건너뛰면 즉시 퇴장당함 |
| 핵심 장면 2 | Mere 없이는 추장에게 접근조차 불가능하다는 걸 깨달음 |
| 플레이어 선택 | Mere에게 주도권을 줄 것인가 vs 내가 직접 진행할 것인가 |
| 챕터 결말 | Mere에게 주도권을 넘기면 → 추장 신뢰 획득. 직접 하면 → 부분 실패, 재시도 필요 |
| 메시지 | 주도권을 내려놓는 것이 협력의 시작 |

---

### 4장 — 나이탬바 섬: "각자도생의 유혹"
- **장소**: 외딴 섬, 해안가 마을
- **시간**: 25분
- **등장 NPC**: Lani (메인), James (갈등), Mere (전화)

| 구분 | 내용 |
|------|------|
| 상황 | 섬에 도착했지만 주민들은 무관심. James가 "우리 기관이 단독으로 하겠다"고 통보. |
| 목표 | Lani의 신뢰 획득, James 설득 |
| 핵심 장면 1 | Lani 인터뷰: "10년 전에도 누군가 왔어요. 발전기 놓고 갔는데 2년 만에 고장났어요. 아무도 안 왔어요." |
| 핵심 장면 2 | James 압박: "우리 예산으로 빠르게 끝낼 수 있어. 당신네 절차가 문제야." |
| 각자도생 선택지 | James 제안 수락 → 빠르게 진행되지만 Lani/Mere/Timoci 신뢰도 폭락 → 배드엔딩 경로 |
| 올바른 경로 | James에게 공동 성과 공유 제안 + Lani에게 이번엔 다르다는 걸 행동으로 보여줌 |
| 챕터 결말 | 폭풍 예보 — 모든 것을 수바에서 결정해야 함 |
| 메시지 | 빠른 길이 항상 옳은 길은 아니다 |

---

### 5장 — 수바: "우리가 이룬 것"
- **장소**: KODA 사무소 회의실 → 나이탬바 섬 (결말)
- **시간**: 20분
- **등장 NPC**: 5인 전원

| 구분 | 내용 |
|------|------|
| 상황 | 5명을 한 자리에 모아 최종 합의를 이끌어내야 함 |
| 목표 | 신뢰도 조건 충족 + 최종 사업 계획 합의 |
| 핵심 장면 | 각자가 왜 협력해야 하는지 직접 말하는 장면 — Lani의 한마디가 분위기를 바꿈 |

**엔딩 조건**
- 진엔딩: 5인 모두 신뢰도 70 이상 → 1년 후 나이탬바 섬. Lani가 웃으며 전등 스위치를 켠다.
- 노멀엔딩: 3~4인 70 이상 → 사업 완료, 하지만 누군가의 빈자리가 느껴짐
- 배드엔딩: 2인 이하 또는 James 단독 선택 → "또 다른 고장난 발전기가 될 수도 있습니다"

**크레딧**: 실제 KOICA 피지 사업 소개 + ODA 정보 링크

---

## 신뢰도 시스템

```
각 NPC 신뢰도: 0 ~ 100

진엔딩:   5인 모두 70 이상
노멀엔딩: 3~4인 70 이상
배드엔딩: 2인 이하 OR James 단독 선택 경로
```
