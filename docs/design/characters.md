## 주인공

| 항목 | 내용 |
|------|------|
| 국적 | 한국인 |
| 성별 | 비공개 (이름/대사에서 성별 드러나지 않게) |
| 직급 | 중간급 (입사 5~10년차) |
| 소속 | KODA (Korea Overseas Development Agency, 가상) |
| 상황 | 피지 사무소 첫 발령 |
| 이름 | 플레이어 직접 입력 |

### 내면 여정
```
처음: "내가 전문가니까 내가 방향 잡으면 된다"
중간: "왜 다들 비협조적이지?"
전환점: 현지 파트너 없이는 자신이 아무것도 모른다는 걸 깨달음
끝: 진짜 협력이 뭔지 이해한 사람이 됨
```

## 핵심 NPC 5인

### NPC 1 — Timoci Naevo
- **소속**: 피지 국가계획부
- **한 줄 성격**: "절차가 곧 나다"
- **겉모습**: 항상 정장, 서류 가방
- **동기**: 자리 보전, 실수 없이 임기 마치기
- **초기 태도**: 형식적, 느림, 공문 요구
- **숨겨진 면**: 사실 현장 문제를 누구보다 잘 알지만 혼자 결정할 용기가 없음
- **협력 전환점**: 주인공이 그의 체면을 살려주는 방식으로 접근할 때
- **일반인 공감 포인트**: "저런 상사 직장에 꼭 있지"
- **등장 챕터**: 2장 (수바 정부청사)

---

### NPC 2 — Ratu Josefa
- **소속**: 나우소리 외곽 마을 추장
- **한 줄 성격**: "우리 마을은 내가 지킨다"
- **겉모습**: 전통 의상, 위엄 있는 태도
- **동기**: 마을 공동체와 전통 보호
- **초기 태도**: 세부세부(Sevusevu) 의식 없이는 대화 거부
- **숨겨진 면**: 젊은 세대가 마을을 떠나는 것에 깊이 상처받음
- **협력 전환점**: 주인공이 전통을 진심으로 존중한다는 걸 보여줄 때
- **일반인 공감 포인트**: "내 동네, 내 방식" — 누구나 자기 영역에선 이렇게 느낌
- **등장 챕터**: 3장 (농촌 커뮤니티)

---

### NPC 3 — Mere Vuniwai
- **소속**: 현지 NGO "Pacific Roots" (가상)
- **한 줄 성격**: "외국 원조는 또 다른 식민지다"
- **겉모습**: 캐주얼, 현장감 있는 차림
- **동기**: 진짜 주민 삶 개선, 외부 의존 탈피
- **초기 태도**: KODA에 냉소적, 불신
- **숨겨진 면**: 혼자서 감당하기엔 리소스가 너무 부족해서 지쳐있음
- **협력 전환점**: 주인공이 주도권을 Mere에게 넘길 때
- **일반인 공감 포인트**: 열정적이지만 번아웃 직전인 활동가
- **등장 챕터**: 1장 (수바 사무소) 첫 만남, 3장에서 핵심 역할

---

### NPC 4 — James Hartley
- **소속**: 가상 호주 기관 "APAT" (Australia Pacific Aid Trust)
- **한 줄 성격**: "우리가 이미 하고 있어요"
- **겉모습**: 세련됨, 자신감 넘침
- **동기**: 자기 기관 성과 극대화, 예산 확보
- **초기 태도**: 겉으론 협조적, 실제론 경쟁적
- **숨겨진 면**: 본국 예산 삭감 압박으로 성과에 쫓기고 있음
- **협력 전환점**: 주인공이 경쟁 대신 성과 공유를 제안할 때
- **일반인 공감 포인트**: "직장에서 만나는 타 부서 사람"
- **등장 챕터**: 2장 조연, 4장 (위기) 핵심 역할

---

### NPC 5 — Lani Taukei
- **소속**: 야사와 군도 외딴 마을 주민
- **한 줄 성격**: "또 조사하러 왔나요?"
- **겉모습**: 수수한 차림, 피곤한 눈빛
- **동기**: 그냥 평범하게 살고 싶음
- **초기 태도**: 무관심, 냉소 ("어차피 바뀌는 거 없어요")
- **숨겨진 면**: 과거 원조 프로젝트가 실패한 걸 직접 경험함
- **협력 전환점**: 처음으로 자신의 의견이 실제로 반영되는 경험을 할 때
- **일반인 공감 포인트**: 가장 강력한 감정선 — 게임의 진짜 결말을 이끄는 캐릭터
- **등장 챕터**: 4장 (외딴 섬), 5장 최종 씬

## 5인 관계 구조

```
주인공
  ├── Timoci   (제도/권한)
  ├── Ratu Josefa  (커뮤니티 접근권)
  ├── Mere     (현장 실행력)
  ├── James    (자원/조율)
  └── Lani     (사업의 존재 이유)

→ 5명 모두 연결되어야 최종 사업 완료
→ 한 명이라도 소외되면 엔딩 조건 미달
```

## 게임 핵심 메시지

> 우리가 협력하면 뭔가 이룰 수 있다.

- 주인공 혼자서는 풀 수 없는 문제들만 존재하도록 설계
- 혼자 처리하는 선택지도 존재하지만 장기적으로 더 큰 문제 발생
- 모든 이해관계자 신뢰도가 일정 수준 이상일 때만 진짜 엔딩 도달 가능
